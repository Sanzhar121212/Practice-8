# Quest Master

Приложение на **PyQt6** для генерации квестов с GUI, экспортом документов и редактором карт.

## Возможности

- Создание квестов:
  - название, сложность, награда, описание, дедлайн;
  - автосохранение в **SQLite**;
  - валидация полей, подсветка ошибок;
  - горячая клавиша **Ctrl+Enter** для создания квеста.

- Шаблоны и экспорт:
  - HTML-шаблоны на **Jinja2** (`royal_decree.html`, `guild_contract.html`, `ancient_scroll.html`);
  - экспорт в **PDF** (WeasyPrint) и **DOCX** (python-docx);

- Редактор карты:
  - холст 800×600, фон «пергамент»;
  - кисть, маркеры локаций (город/логово/таверна), текстовые метки;
  - сохранение карты в PNG/JPG, загрузка фона;
  - локации привязаны к квесту в БД.

- Геймификация:
  - уровни и XP;
  - прогресс-бар и список достижений;
  - звуковой эффект при получении XP.

## Требования

- Python 3.10+
- PyQt6  
- Jinja2  
- weasyprint  
- python-docx  
- qrcode[pil] (опционально)  
- pytest (для теста «босс-файт»)
